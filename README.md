# TOTALOG
TOTALOG is a simple FLOSS Ham Radio logging app for small contests and portable operation. 

Features include:
* logging QSOs
* edit logged QSOs afterwards
* archive and manage different QSO lists
* export and share QSO lists as cabrillo
* dark mode
